

## הוספת Sitelinks בתוצאות חיפוש גוגל

### מה זה Sitelinks?
Sitelinks הם הלינקים הנוספים שמופיעים מתחת לתוצאת החיפוש הראשית של אתר בגוגל. גוגל מחליט אלגוריתמית אילו לינקים להציג, אבל יש דברים טכניים שנוכל לעשות כדי לשפר משמעותית את הסיכוי שיופיעו.

### מה כבר קיים באתר (ועובד טוב)
- Schema של SiteNavigationElement עם כל הדפים
- SSR Fallback עם לינקים סטטיים
- Breadcrumbs עם BreadcrumbList schema בכל דף פנימי
- Footer עם לינקים לכל הדפים
- כותרות ייחודיות ותיאורים לכל דף

### הבעיה המרכזית
**הניווט הראשי מוסתר בתפריט המבורגר** - גם בדסקטופ. גוגל מעדיף בצורה חזקה לינקים גלויים בתוך אלמנט `<nav>` ולא לינקים חבויים מאחורי כפתור. זו הסיבה העיקרית שכנראה גוגל לא מציג Sitelinks.

### השינוי המוצע

הוספת ניווט דסקטופי גלוי (Desktop Navigation Bar) שיופיע במסכים רחבים, מתחת להדר הקיים או כחלק ממנו.

#### שלב 1: הוספת Desktop Nav ב-PersistentHeader

הוספת שורת ניווט גלויה שמופיעה רק בדסקטופ (`hidden lg:flex`), עם לינקים ישירים לכל דפי האתר.

**קובץ: `src/components/PersistentHeader.tsx`**

- הוספת import של `Link` מ-react-router-dom
- הוספת מערך `navLinks` עם כל הדפים
- הוספת שורת `<nav>` מתחת להדר הקיים שמופיעה רק במסכים מעל `lg`
- הלינקים יהיו בסגנון קומפקטי ונקי שמשתלב עם העיצוב הקיים

מבנה מוצע:

```text
+----------------------------------------------------------+
| [Logo]                    [CTA Button] [Hamburger(mobile)]|
+----------------------------------------------------------+
| דף הבית | התקנה | גדר סולארית | לבית | מחירון | למה? | אזורים | בלוג |  <- חדש, רק דסקטופ
+----------------------------------------------------------+
```

#### שלב 2: עדכון SSR Fallback ב-index.html

הוספת אלמנט `<nav>` גלוי גם בתוכן הסטטי של ה-SSR fallback, עם לינקים ישירים (לא רק בסקציה של "השירותים שלנו" אלא כחלק מהדר).

**קובץ: `index.html`**

- הוספת שורת `<nav>` עם לינקים לכל הדפים בחלק העליון של ה-SSR fallback
- זה מבטיח שגם בוטים שלא מריצים JavaScript רואים את הניווט

---

### פרטים טכניים

**קבצים שישתנו:**

| קובץ | שינוי |
|---|---|
| `src/components/PersistentHeader.tsx` | הוספת שורת ניווט דסקטופית גלויה עם Link components |
| `index.html` | הוספת nav element בראש ה-SSR fallback |

**עקרונות:**
- הניווט הדסקטופי יהיה `hidden lg:flex` - מוסתר במובייל, גלוי בדסקטופ
- תפריט ההמבורגר ישאר למובייל כרגיל
- הלינקים ישתמשו ב-`<Link>` של React Router
- העיצוב ישתלב עם הסגנון הקיים של האתר (צבעי primary, טיפוגרפיה)
- ה-SSR fallback יכיל את אותם לינקים כ-`<a>` tags רגילים
